# 🌱 Sprout Handong

한동대학교 RC(Residential College) 학생들을 위한 커뮤니티 플랫폼

## 📋 프로젝트 소개

Sprout Handong은 같은 RC 학생들끼리 추억을 공유하고 소통할 수 있는 웹 플랫폼입니다.

### 주요 기능

- **회원가입 & 로그인**: 간편한 계정 관리
- **프로필 설정**: 닉네임, RC, 팀 설정
- **새싹게시판**: 같은 RC 학생들과 게시글 공유
- **삼행시**: 재미있는 삼행시 작성 및 공개/비공개 설정
- **추억 달력**: RC의 특별한 날들을 기록
- **앨범**: 소중한 순간을 사진으로 저장
- **팀 기능**: 팀을 만들어 비공개 삼행시 공유

## 🚀 사용 방법

### 1. 서버 실행하기

#### 방법 1: npm으로 실행
```bash
cd login-board
npm start
```

#### 방법 2: Node.js로 직접 실행
```bash
cd login-board
node server.js
```

서버가 성공적으로 실행되면 다음 메시지가 표시됩니다:
```
서버가 http://localhost:3000 에서 실행중입니다.
```

### 2. 웹사이트 접속하기

#### 내 컴퓨터에서 접속
브라우저를 열고 다음 주소로 접속하세요:

```
http://localhost:3000
```

#### 같은 와이파이를 쓰는 다른 기기에서 접속
1. **내 컴퓨터 IP 주소 확인**:
   ```bash
   # Windows 명령 프롬프트에서
   ipconfig

   # "IPv4 주소" 찾기 (예: 192.168.0.10)
   ```

2. **다른 기기에서 접속**:
   ```
   http://192.168.0.10:3000
   (내 IP 주소로 바꿔서 입력)
   ```

#### localhost와 3000은 무엇인가요?
- **localhost**: 내 컴퓨터를 가리키는 특별한 주소 (127.0.0.1)
- **3000**: 서버가 사용하는 포트 번호 (프로그램의 통신 채널)

### 3. 회원가입 & 로그인

1. **회원가입**: 메인 화면에서 "회원가입" 링크 클릭
   - 아이디, 비밀번호, 이름 입력

2. **로그인**: 가입한 아이디와 비밀번호로 로그인

3. **프로필 설정**: 최초 로그인 시
   - 닉네임 입력
   - RC 선택 (토레이 College, 손양원 College, 카이퍼 College, 열송학사, 장기려 College, 카마이클 College)
   - 팀 생성 또는 기존 팀 참여

## 📱 주요 페이지 안내

### 홈 (home.html)
- 최근 게시글 및 삼행시 슬라이더
- 각 기능으로 빠른 이동

### 새싹게시판 (board.html)
- 같은 RC 학생들의 게시글 확인
- 게시글 작성, 좋아요, 삭제
- 삼행시 작성 (공개/비공개 선택 가능)

### 추억 달력 (calendar.html)
- 기념일 추가 및 관리
- D-day 카운터

### 앨범 (album.html)
- 사진 업로드 및 관리
- 로컬 저장소에 저장

### 설정 (settings.html)
- 닉네임 변경
- 이모티콘 설정
- 팀 관리
- RC 변경 (24시간 쿨다운)

## 🛠️ 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Data Storage**: JSON 파일 기반
- **Dependencies**:
  - express (^5.2.1)
  - body-parser (^2.2.1)

## 📂 프로젝트 구조

```
login-board/
├── server.js           # 서버 메인 파일
├── package.json        # 프로젝트 설정
├── data/              # 데이터 저장 폴더
│   ├── users.json     # 사용자 정보
│   ├── posts.json     # 게시글 정보
│   ├── acrostics.json # 삼행시 정보
│   └── teams.json     # 팀 정보
└── public/            # 프론트엔드 파일
    ├── index.html     # 로그인 페이지
    ├── signup.html    # 회원가입 페이지
    ├── nickname.html  # 프로필 설정 페이지
    ├── home.html      # 홈 페이지
    ├── board.html     # 게시판 페이지
    ├── calendar.html  # 추억 달력 페이지
    ├── album.html     # 앨범 페이지
    └── settings.html  # 설정 페이지
```

## ⚠️ 주의사항

1. **서버가 실행 중이어야 합니다**: 웹사이트를 사용하려면 반드시 서버를 먼저 실행해야 합니다.

2. **포트 3000 사용**: 기본적으로 3000 포트를 사용합니다. 다른 포트를 사용하려면 환경변수 PORT를 설정하세요.

3. **데이터 저장**: 모든 데이터는 `data` 폴더의 JSON 파일에 저장됩니다. 백업이 필요한 경우 이 폴더를 복사하세요.

4. **RC 변경 제한**: RC 변경은 24시간에 한 번만 가능합니다.

5. **팀 해체**: 팀장이 팀을 나가면 팀이 해체되고 모든 팀원이 자동으로 탈퇴됩니다.

## 🔒 보안 참고사항

현재 버전은 교육 및 데모 목적으로 제작되었습니다. 실제 프로덕션 환경에서 사용하려면 다음 사항을 개선해야 합니다:

- 비밀번호 암호화 (bcrypt 등)
- JWT 기반 인증
- HTTPS 적용
- 입력 값 검증 강화
- SQL Injection, XSS 방어 강화

## 📞 문제 해결

```bash
# 의존성 재설치
npm install

# 다시 실행
npm start
```

- 서버가 실행 중인지 확인
- 브라우저 주소가 `http://localhost:3000`인지 확인
- 브라우저 캐시 삭제 후 재시도
